# what-is-agent 项目上下文

## 项目目的

本项目旨在写一篇 **面向非计算机专业爱好者** 的中文文档，high level 地介绍 AI Agent 是什么、怎么工作、有哪些常见模式和陷阱，帮助读者对 Agent 技术祛魅、建立基本认知，同时避免踩坑。

## 写作规则

- 使用中文编写。除非特别注明，所有内容（包括代码注释、说明文档等）均以中文为准。
- 文档以网页的形式保存在`index.html`，自包含，不依赖任何外部项，可以离线打开。
- 这是一篇走马观花式的迷你文档。篇幅宜短不宜长，行文高度概括、点到为止，让读者可以快速扫完。
- 目标是传递"其实没什么复杂东西"的祛魅效果，而非事无巨细的科普文章。
- 面向零技术背景的读者，避免使用专业术语，必须使用时需先通俗解释。
- 不要纠结于细节，只关注核心概念，不要涉及琐碎的技术细节和边界条件。
- 内容风格：通俗易懂、举例说明、少讲公式多讲直觉。
- 借助图表辅助说明，使用Cavas绘制。
